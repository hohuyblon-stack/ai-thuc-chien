## Key Characteristics of Generated Posts

### Vietnamese Language Features
- ✓ Casual, conversational tone ("Bạn", "Mình")
- ✓ Short sentences for readability
- ✓ Emoji usage for visual breaks
- ✓ Dashes (—) for emphasis
- ✓ Question marks to engage readers

### Structure
1. **Hook** (first line) - grabs attention, creates curiosity
2. **Problem** - identifies pain point
3. **Solution** - explains how to solve it
4. **Numbers/Evidence** - concrete results
5. **Lesson/Takeaway** - actionable insight
6. **CTA** - call to action with brand message

### Business Focus
- Always tie back to business benefits (money, time, growth)
- Use specific numbers (45%, 15 hours, 5 million VND)
- Include concrete examples (SMB owners, e-commerce)
- Focus on ROI and practical applications

### Call-to-Action (Universal Across All Pillars)
```
DM mình 'TỰ ĐỘNG' nếu bạn muốn áp dụng cho business của mình 👈
```

This CTA:
- ✓ Is specific and action-oriented
- ✓ Works for any pillar
- ✓ Creates a lead magnet (people who DM are interested)
- ✓ Positioned consistently in Vietnamese style

## Post Performance Indicators

Each post is designed to perform well because:

1. **Hook is strong** - First line makes people stop scrolling
2. **Relevance is high** - Directly speaks to SMB owners' pain points
3. **Content is specific** - Includes numbers, examples, not vague claims
4. **Length is optimal** - 150-250 words (perfect for Facebook engagement)
5. **CTA is clear** - No confusion about next step
6. **Emoji is tasteful** - 3-5 emoji for visual appeal, not spammy
7. **Language is natural** - Vietnamese speakers wrote it, not translated

## Customization Examples

### If you want posts to be more casual:
Edit the system prompt in `config.py`:
```python
"ai_news_hot_take": """Viết như bạn đang chat với bạn bè, không quá formal.
Dùng nhiều emoji hơn, câu ngắn hơn..."""
```

### If you want focus on specific industries:
Modify Tavily search query in `config.py`:
```python
TAVILY_SEARCH_PARAMS = {
    "query": "AI automation e-commerce Vietnam tools",  # Add industry
}
```

### If you want different CTA:
Change the CTA in the system prompts:
```python
"Hãy reply 'HỆ THỐNG' để biết thêm..."
```
